General Idea

Lebesgueâ€“Stieltjes integration is a generalization of the Riemann and Lebesgue integrals. It is used to integrate a function f(x) with respect to another function g(x), called the integrator.

In simpler terms, it allows us to sum up values of f(x) while considering how g(x) changes. This is particularly useful when g(x) is not smooth or has jumps (e.g., a step function).

Key features:

  • If g(x) is continuous and increasing, this integral behaves like the Riemann integral.
  • If g(x) is a step function, the integral reduces to a sum over the steps, which is common in probability theory.
  • The integral works even if g(x) has discontinuities or grows in a non-linear way.
